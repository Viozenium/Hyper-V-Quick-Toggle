# Hyper-V Quick Toggle

## Italiano

Questo script Python permette di visualizzare rapidamente lo stato di Hyper-V (attivato o disattivato) e di modificarlo con un semplice pulsante.

Descrizione:
Lo script verifica lo stato corrente di Hyper-V e consente di abilitarlo o disabilitarlo senza dover digitare manualmente i comandi PowerShell.
È pensato per chi ha la necessità di passare frequentemente tra Hyper-V attivo e disattivato.

Nota: La modifica dello stato non viene applicata automaticamente in caso di crash o errori imprevisti.

Requisiti:
1) Esecuzione dello script con permessi di amministratore, necessari per l’applicazione delle modifiche.
2) Riavvio del computer dopo ogni cambio di stato per rendere effettiva l’impostazione.

Motivazione:
Questo script nasce dalla necessità di attivare e disattivare Hyper-V rapidamente, evitando di dover inserire ogni volta manualmente i comandi in PowerShell

---

## English

This Python script allows you to quickly view the status of Hyper-V (enabled or disabled) and change it with a simple button.

Description:
The script checks the current Hyper-V status and allows you to enable or disable it without manually typing PowerShell commands.
It is designed for users who frequently need to switch between Hyper-V being enabled or disabled.

Note: The status change is not applied automatically in case of crashes or unexpected errors.

Requirements:
1) Run the script with administrator privileges, which are necessary to apply the changes.
2) Restart the computer after each status change to make the setting effective.

Motivation:
This script was created to quickly enable and disable Hyper-V, avoiding the need to manually enter PowerShell commands each time.
