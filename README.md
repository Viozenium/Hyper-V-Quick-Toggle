# Hyper-V-Quick-Toggle
Uno script Python che permette di visualizzare e modificare rapidamente lo stato di Hyper-V su Windows. Consente di attivare o disattivare Hyper-V con un solo clic, evitando di digitare manualmente comandi PowerShell. Richiede privilegi amministrativi e un riavvio del sistema per applicare le modifiche.
